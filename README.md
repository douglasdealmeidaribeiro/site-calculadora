# Calculadora de Financiamento Price

Esta é uma aplicação web simples para calcular parcelas mensais de um financiamento no sistema Price e exibir a tabela completa de amortização.

## Como usar

Abra o arquivo [public/index.html](public/index.html) em um navegador ou suba o projeto em um servidor estático.

## Funcionalidades

- Entrada do valor financiado
- Número de meses
- Taxa de juros mensal
- Cálculo da parcela mensal
- Exibição da tabela de amortização completa

## Publicação no GitHub Pages

1. Acesse o repositório no GitHub.
2. Vá em Settings > Pages.
3. Selecione a branch `main` e a pasta `/root`.
4. Salve.
